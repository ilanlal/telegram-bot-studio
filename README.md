# ![Logo](https://raw.githubusercontent.com/ilanlal/telegram-bot-studio/main/assets/google-workspace-marketplace/24x24.png) Telegram™ Bot API Studio

This add-on allows you to integrate Google Sheets™ with Telegram™ Bot API, enabling you to manage and manipulate your bot's data directly from Google Sheets™.

> **Note**: This add-on is currently in development. Some features are planned but not yet implemented. Please refer to the "Plugins" section below for details.

## Plugins

![Logo](https://raw.githubusercontent.com/ilanlal/telegram-bot-studio/main/assets/google-workspace-marketplace/1376x768_Gemini_Generated_Image_30sdhg30sdhg30sd.png)
The add-on is designed with a plugin architecture, allowing for easy expansion and customization. Each plugin corresponds to a specific set of functionalities related to the Telegram Bot API.

Current plugins include:

- **Get Me**: Retrieve basic information about the bot.
- **Get Chat**: Fetch details about a specific chat.
- **Webhook**: Set up and manage webhooks for real-time updates.

Future plugins planned:

- **sendInvoice**: Send invoices to users.
- **createInvoiceLink**: Generate invoice links for payments.

Gift-Related Features (Planned):

- **sendGift**: Send gifts to users or chats.
- **getAvailableGifts**: List available gifts for sending.
- **getUserGifts**: Retrieve gifts sent to the bot by users.
- **getChatGifts**: Fetch gifts associated with a specific chat.
- **transferGifts**: Transfer gifts between users or chats.

Business Account Gift Features (Planned):

- **setBusinessAccountGiftSettings**: Configure gift settings for business accounts.
- **getBusinessAccountStarBalance**: Check the star balance of business accounts.
- **getBusinessAccountGifts**: Retrieve gifts sent to business accounts.
- **transferBusinessAccountStars**: Transfer stars between business accounts.
